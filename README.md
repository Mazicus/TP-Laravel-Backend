# TP Laravel Backend

A simple Laravel backend project.

## About

This is a backend built with Laravel as part of a personal project.
Nothing fancy — just clean and functional.

## Requirements

- PHP >= 8.0
- Composer
- MySQL or any supported database

## Installation

```bash
git clone https://github.com/Mazicus/TP-Laravel-Backend.git
cd TP-Laravel-Backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```

## Usage

Once running, the API will be available at `http://localhost:8000`
